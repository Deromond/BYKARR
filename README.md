# BYKARR

Hybrid Android application with native and WebView-based flow.

## Stack
- Kotlin
- XML
- WebView
- Coroutines
- OneSignal
- AppsFlyer

## Features
- Custom launch / preloader flow
- Remote config via GitHub Gist
- WebView-based entry logic with fallback
- Attribution and conversion tracking
- Push notifications integration

## What I implemented
- Entry flow and decision logic (native vs WebView)
- Attribution engine and dynamic URL generation
- Error handling and fallback scenarios
- Native business logic screens
